Configuración inicial
Una sola vez · Desde computadora · ~5 min
1
Entrá a console.firebase.google.com con tu Gmail.
2
Crear proyecto → nombre (ej: stock-manager) → desactivá Analytics → Crear.
3
Authentication → Comenzar → Google → Activar. Poné tu correo de soporte y guardá.
4
Firestore Database → Crear base de datos → Modo producción → Región us-east1.
5
En Firestore → Reglas, pegá esto y reemplazá los correos de las 2 personas:
rules_version = '2'; service cloud.firestore { match /databases/{db}/documents { match /{doc=**} { allow read, write: if request.auth.token.email in [ 'correo1@gmail.com', 'correo2@gmail.com' ]; } } }
⚠️ Reemplazá los correos exactos y hacé click en Publicar.
6
⚙️ Configuración → Tus apps → Web </> → registrá la app → copiá el bloque firebaseConfig.
7
Abrí este archivo con el Bloc de notas, buscá TU_API_KEY con Ctrl+F y reemplazá el bloque completo. Guardá.
✓ Abrí en Chrome y probalo. Compartí el archivo por WhatsApp o Drive con las 2 personas.
